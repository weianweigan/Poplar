# Poplar

A revit addin to distinguish and statistics busbar in your *.rvt file.

Support Element Type:
```csharp
    BuiltInCategory.OST_DuctCurves
    BuiltInCategory.OST_DuctFitting
```

You can also select some elements to distinguish.
